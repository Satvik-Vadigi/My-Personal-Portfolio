# 🚀 My Personal Portfolio

Live Demo 👉 https://my-personal-portfolio-virid-xi.vercel.app/

---

## 📌 Overview

This is my personal developer portfolio built using modern full-stack technologies.  
It showcases my projects, skills, experience, and includes a working contact system.

The goal of this project is to demonstrate **real-world development skills**, clean UI/UX, and production-ready deployment.

---

## ⚙️ Tech Stack

- ⚡ Next.js 16 (App Router)
- 🎨 Tailwind CSS
- 🎬 Framer Motion (animations)
- 🌗 Dark/Light Theme (next-themes)
- 📩 Nodemailer (Email API)
- ☁️ Vercel (Deployment)

---

## ✨ Features

- 🔥 Modern responsive UI (mobile + desktop)
- 🎬 Smooth animations & transitions
- 🌗 Dark / Light mode toggle
- 📂 Projects showcase with live & GitHub links
- 📊 Skills & experience section
- 📩 Functional contact form (sends email)
- 🚀 Fully deployed on Vercel
